NACHC-CAD SMART on FHIR Cener Example Application

This application is the demo application for a SMART on FHIR Application run on the Cerner Sandbox.

This application can be built using Maven (MVN) and deployed to your favorite Java enabled Web Server (e.g. Tomcat).

To deploy this application to Cerner's SMART on FHIR Sandbox you can run the application on your localhost and use the following urls:

  • Redirect URI:
    http://localhost/nachc-smart-on-fhir-cerner-example/example-smart-app/
  • SMART Launch URI:
    http://localhost/nachc-smart-on-fhir-cerner-example/example-smart-app/launch.html

This application has been deployed to the Cerner Sandbox as NACHC-CAD Example using localhost
